# C语言的信息管理系统
一个用于课程实践的 C语言 命令行职工信息管理系统，支持多角色权限、链表数据存储与文件持久化。
> A simple staff information management system used for school work

## 仓库介绍
这是一份C语言编写的职工信息管理系统的课程作业，采用小熊猫C++IDE编写。该系统功能相对全面，代码量较大，但没有图形化界面，是在终端中进行交互的程序。另外为了修改数据的方便，虽然本程序使用.dat文件存储信息，但是依然是格式化写入，可以用记事本打开直接修改数据。编译使用推荐使用小熊猫C++，不容易出现问题。目的帮助完全没有思路的学生提供一种思路，这份思路不是最好甚至可能不是较好，可能存在不少问题，请自行辨析。

## 这类系统的常见要求
- [x] 1.基本的增删改查
- [x] 2.使用链表存储数据
- [x] 3.借助文件读写进行数据的持久化
- [x] 4.分角色划分权限
- [x] 5.针对特定系统的要求：比如职工信息管理系统通常会要求薪资信息的处理

## 开发环境
- 系统：windows10
- IDE：redpandaC++ 3.3.3147
- 编译器：MinGW-w64 GCC 11.0.5

## 文件内容
- **staff_info_manage_system.c**：
  - 分为管理员，职工，经理三种角色；
  - 数据文件读写使用两个文件staff_info.dat和user_info.dat；
  - 包括增删改查和薪资相关的函数
- staff_info.dat：职工数据存储形式
- user_info.dat：用户数据存储形式
- student_info_manage_system.c：附加一个极简的学生管理系统，只有最基本增删改查和读取保存的功能
