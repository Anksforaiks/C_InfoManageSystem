# C语言的信息管理系统
A simple staff information management system used for school work

## 仓库介绍
这是一份C语言编写的职工信息管理系统的课程作业，采用小熊猫C++IDE编写。该系统功能相对全面，代码量较大，在编写的工程中有使用多种AI辅助。主要作业是希望
可以帮助完全没有思路的学生提供一种思路，这份思路不是最好甚至可能不是较好，可能存在不少问题，请自行辨析。本仓库代码只推荐用于学习借鉴，不希望完全复制粘贴使用。

## 这类系统的常见要求
* 1.基本的增删改查
* 2.使用链表存储数据
* 3.需要借助文件读写进行数据的持久化
* 4.针对特定系统的要求：比如职工信息管理系统通常会要求薪资信息

## 文件内容
- **staff_info_manage_system.c**：
  - 分为管理员，职工，经理三种角色；
  - 数据文件读写使用两个文件staff_info.dat和user_info.dat；
  - 包括增删改查和薪资相关的函数
- staff_info.dat：职工数据存储形式
- user_info.dat：用户数据存储形式
